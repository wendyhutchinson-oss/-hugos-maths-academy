# Hugo's Maths Academy

An interactive Key Stage 2 maths revision website for Hugo.

## Files
- `index.html` — main website page
- `style.css` — design and layout
- `script.js` — questions, scoring, XP and progress saving

## How to publish with GitHub Pages
1. Upload these files to the repository root.
2. Go to Settings.
3. Go to Pages.
4. Under "Build and deployment", choose "Deploy from a branch".
5. Branch: `main`
6. Folder: `/root`
7. Save.
